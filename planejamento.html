Conectar ao planejamento

Cole seu token do GitHub para carregar e salvar o planejamento (sincroniza entre PCs).

Como criar o token (1 min)
  1. Abra github.com/settings/personal-access-tokens/new
  2. Em Repository access → Only select repositories → escolha aurora-planejamento
  3. Em Permissions → Repository permissions → Contents → Read and write
  4. Clique Generate token, copie e cole aqui
